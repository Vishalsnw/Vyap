# InvoicePro - GST Billing Automation

## Overview
This project contains specifications for an Invoice + GST Billing Automation application for small businesses in India. Originally designed as an Android app, we've created a web-based version that runs in the browser.

## Project Structure
- `Vyapar.txt` - Original requirements document
- `index.html` - Main web application (single-page app)

## Features
- Business profile setup
- Customer management
- Product/service management with GST rates
- Invoice creation with automatic GST calculation
- PDF invoice generation
- Dashboard with sales summary

## Technical Details
- Frontend: HTML, CSS, JavaScript
- Storage: Browser localStorage (offline-first)
- PDF Generation: jsPDF library

## Recent Changes
- 2026-01-15: Initial web version created for Replit environment

## Notes
The original spec was for a Kotlin Android app. This web version provides equivalent functionality that can run in any browser.
